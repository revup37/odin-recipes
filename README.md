# odin-recipes
I created a site with multiple chicken recipes. This has 3 chicken recipes, 1 of which is one of the highest rated chicken recipe in the allrecipes website.

With this exercise, I managed to create a main landing page with 3 different links for each recipe.
With each recipe, I created a header,  added an image with links, and even  modified the image size to be uniform with each recipe links.

The exercise also taught me to create multiple lists, both ordered and unordered to showcase the ingredients and directions on how to make the recipe.
